# Vue (pronounced /vjuː/, like view)

--
```
<div id="app">
  <input type="text">
  <div></div>
</div>
```
--

```
<div id="app">
  <input type="text" v-model="message">
  <div>{{ message }}</div>
</div>
```
--
```
new Vue({
  el: '#app',
  data: {
    message: 'BeerJS is awesome'
  }
})
```
